Activity 5 Description

This C++ program allows the user to input the number of players (up to 4) and their scores, then sorts the players by their scores in descending order using a simple sorting method. It first asks for the number of players and adjusts if needed, then updates the players' scores. Afterward, it sorts the scores and displays the players' rankings from highest to lowest score, showing each player's placement and score.
